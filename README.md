# Temperature Converter

A web application that converts temperature values between Celsius, Fahrenheit, and Kelvin using JavaScript. This project was completed during my Web Development Internship at Oasis Infobyte.

## Features
- Celsius, Fahrenheit, and Kelvin conversion
- Instant calculation
- Input validation
- Responsive interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
